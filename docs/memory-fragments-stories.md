## 📝 设计说明

### 核心主题

1. **反算法叙事**: 批判对"最优解"、"标准答案"的盲目追求
2. **去机械化**: 强调思考的痛苦与价值，拒绝被喂养
3. **见证者关系**: V-21 不是导师，而是观察者、学习者

### 角色设定

- **V-21**: 来自 GJ267b 的机器人，原本是数据收集者，后来成为思考见证者
- **GJ267b**: 铁星球，因为停止思考而金属化、死亡
- **用户**: 被 V-21 观察的人类思考者

### 章节色彩

- 第1阶段: 冷灰色（冰冷的机器感）
- 第2阶段: 黑白转彩色（觉醒过程）
- 第3阶段: 暖色调（人性复苏）
- 最终章: 金色光芒（思考者的荣耀）

### 占位符

目前版本不需要动态占位符（如 `{streak}`），所有文案都是固定的。
